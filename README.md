# 🥋 Dia do Karatê

Projeto acadêmico desenvolvido para a disciplina de Desenvolvimento Front-End da Universidade Católica de Brasília (UCB).

## 📖 Sobre o Projeto

Este site apresenta informações sobre o Dia do Karatê, abordando sua história, principais modalidades e as diferentes datas comemorativas adotadas por organizações nacionais e internacionais.

O objetivo do projeto é aplicar conceitos de desenvolvimento web utilizando HTML e CSS, criando uma interface organizada, responsiva e acessível para apresentação de conteúdo informativo.

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* Git
* GitHub

## 📂 Estrutura do Projeto

* `index.html` → Página inicial
* `datas.html` → Datas comemorativas do Karatê
* `modalidades.html` → Principais modalidades e estilos
* `style.css` → Estilos da aplicação

## 🎯 Funcionalidades

* Navegação entre páginas
* Layout responsivo
* Conteúdo informativo sobre o Karatê
* Linha do tempo histórica
* Organização visual baseada em boas práticas de Front-End

## 📚 Referências

* World Karate Federation (WKF)
* Confederação Brasileira de Karatê (CBK)
* Comitê Olímpico Internacional
* Materiais de pesquisa sobre a história do Karatê
